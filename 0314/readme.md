- tree 패키지 설치
```
sudo apt install tree
```

- directory
  + / &emsp;&emsp;: root directory 
  + ~ &emsp;&emsp;: home directory 
  + . &emsp;&emsp; : slef 
  + .. &emsp;&emsp;: parent 

- directory 생성
```
mkdir 디렉토리명
mkdir software
```

- directory 절대경로 이동
```
cd /디렉토리명
cd /home/stud/software
```
- directory 상대경로 이동(현재 디렉토리의 자식 디렉토리)
```
cd ./디렉토리명
cd ./software
```

- file 생성
  + 생성 후에는 파일 내용 입력 가능
  + ctrl+d: 저장 및 종료
```
cat > 생성파일명.파일확장자
cat > sample.txt
```
